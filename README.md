# School Database Schema Project - MIS 443

This repository contains the complete implementation of the **School Database Schema** using PostgreSQL and pgAdmin 4 for the MIS 443 Business Data Management course.

## Original Practice Platform
- **SQL Practice Online:** [School Schema Practice](https://www.sql-practice.online/practice/school?engine=postgresql)

## Tools & Technologies Used
- **Database Management System:** PostgreSQL
- **Database Client:** pgAdmin 4
- **Data Formats:** CSV (Comma-Separated Values)
- **Version Control:** Git & GitHub
